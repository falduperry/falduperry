## Featured projects

### 🧠 GenAI Content Workflow Engine
> OpenAI · Claude API · Adobe Firefly · React · Node.js

Built the AI content pipeline inside Accenture Song's Unified Marketing Platform — integrating 4 LLM/generative AI providers into a single unified workflow used by enterprise marketing teams. Reduced average content creation time by ~50% across 10,000+ monthly sessions.

Key work: prompt engineering architecture, provider orchestration, fallback strategies, cost management, UX feedback loops.

---

### 🏗️ Micro-Frontend Architecture System
> Webpack 5 Module Federation · React · Next.js · AWS Amplify · CI/CD

Designed and implemented a micro-frontend system across 6+ independently deployed applications — used by 4 product squads at Accenture Song. Established shared component libraries, design tokens, and inter-app communication contracts. Cut cross-team delivery time by ~40%.

---

### 📄 Custom Document Export Engine
> React · Node.js · PDF · Word · Chart rendering

Built a from-scratch export engine handling complex document generation — charts, rich text formatting, dynamic layouts — in both PDF and Word formats. Eliminated 3 third-party library dependencies and reduced export processing time by ~60%.

---

### ⚡ Real-Time Collaboration Features
> WebSocket · React · TypeScript · Node.js

Delivered Figma/Google Docs-style real-time collaboration (anchored comments, highlights, discussion threads) across 4 core editor views in an enterprise content platform. Handles concurrent multi-user sessions at scale.
